AlphaText — About


AlphaText is a Chrome extension for increasing the readability of online texts. By adjusting the font and layout of the webpage, you can tweak the text to make it more readable for you.

In AlphaText, you are able to apply individual style settings into a page through Quick Styles, or apply a set of saved styles to a webpage through Profiles. You are also able to save websites to automatically run AlphaText. Experiment with styles and find the best set for you!

Remember to rate and review AlphaText on Chrome Web Store!
